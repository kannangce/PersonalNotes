## Terminal installs
- Brew
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
- jq
```
brew install jq
```
- wget
```
brew install wget
```
- jenv
```
brew install jenv
```
- python
```
brew install python
# python 2 will be available with python2 command
# python 3 will be available with python3 command
```
